  • Nên viết Javascript vào 1 file .js và sử dụng src trong thẻ script để gọi ra trong cuối phần <body> để có hiệu suất tốt nhất.Sỡ dĩ làm như thế để khi trang HTML được load lên thì nội dung sẽ được hiển thị trước và cuối cùng mới tới Javascript. Nếu gọi file .js trong phần đầu <body> file HTML sẽ gọi file Javascript trước. Nếu file Javascript quá lớn sẽ làm chậm quá trình load nội dung , giảm hiệu suất trang web.

Câu 2: Trong Javascript có những kiểu dữ liệu nào ? Làm thế nào để tạo một biến để lưu giá trị tương ứng với những kiểu dữ liệu đó.

  • Kiểu dữ liệu trong Javascript.

    Javascript có các kiểu dữ liệu như sau:

    • Kiểu boolean

    • Kiểu của null

    • Kiểu của undefined

    • Kiểu số

    • Kiểu Symbol (mới trong ECMAScript 6)

  • Để khai báo một biến ta sử dụng từ khóa var <tên biến> ví dụ:

    var username

  • Tuy nhiên không phải muốn đặt tên biến gì cũng được mà phải tuân theo quy tắc sau đây:

    • Tên biến phải là các chữ không dấu viết hoa hoặc viết thường, các chữ số từ 0-9 và dấu gạch dưới (_).

    • Tên biến bắt đầu phải là chữ hoặc dấu gạch dưới (_), nếu bắt đầu bằng số là sai

    • Tên biến có thể đặt dài hay ngắn tùy vào lập trình viên.

Câu 3: Làm thế nào để truy xuất(lấy giá trị) 1 thẻ HTML bất kì theo ID, theo Class, theo Name, theo tên thẻ HTML và theo CSS Selector? Cho ví dụ.

  • Truy xuất theo ID

    var myElement = document.getElementById("intro");

    Try it yourself

  • Truy xuất theo Class

    var x = document.getElementsByClassName("intro");

    Try it yourself

  • Truy xuất theo TagName

    var x = document.getElementsByTagName("p");

    Try it yourself

  • Truy xuất theo CSS Selector

    var x = document.querySelectorAll("p.intro");
